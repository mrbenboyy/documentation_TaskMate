Introduction

TaskMate est une application de gestion de tâches moderne, construite sur le stack MERN (MongoDB, Express.js, React, Node.js). Elle permet aux utilisateurs de créer un compte, se connecter et gérer leurs tâches efficacement.

TaskMate offre deux méthodes pour gérer les tâches : manuellement via l'interface utilisateur ou automatiquement grâce à un chatbot intégré, qui facilite l'organisation des tâches.

Versions Supportées :

Version 1.0.0 : Première version stable de l'application, permettant l'inscription, la connexion et la gestion des tâches.

À propos de TaskMate

TaskMate est conçu pour offrir une solution intuitive et flexible pour la gestion des tâches quotidiennes. Vous pouvez créer des tâches, les organiser et les suivre directement depuis votre tableau de bord. Le chatbot intégré facilite la gestion des tâches de manière plus fluide.

Technologie

TaskMate utilise le stack MERN, composé de :

  • MongoDB : Base de données NoSQL pour stocker les informations des utilisateurs et des tâches.
  • Express.js : Framework backend pour la création des API RESTful.
  • React : Framework frontend pour créer une interface utilisateur moderne et dynamique.
  • Node.js : Environnement d'exécution pour le serveur backend.

Démarrer

Comment démarrer

Installation de l'application

Pour installer TaskMate, suivez ces étapes :

  1. Clonez le dépôt GitHub de TaskMate :
    https://github.com/mrbenboyy/chatbot-task-manager-frontend.git
    https://github.com/mrbenboyy/chatbot-task-manager-backend.git

  2. Installez les dépendances avec npm install dans les répertoires backend et frontend.

  3. Créez un fichier .env pour configurer les variables nécessaires comme la connexion à MongoDB et le port du serveur.

  4. Lancez le serveur backend avec npm start, puis le frontend avec npm start.

Exécution de l'application

Une fois l'application installée, vous pouvez la lancer en accédant aux URLs suivantes :

  • http://localhost:3000 pour le frontend.
  • http://localhost:5050 pour l'API backend.

Vous pourrez alors créer un compte, vous connecter et commencer à ajouter des tâches.

Interfaces de TaskMate

Voici un aperçu des différentes interfaces de TaskMate, qui permet à l'utilisateur de naviguer facilement et de gérer ses tâches efficacement.

1. Page d'Accueil (Home)

La page d'accueil présente un guide sur le fonctionnement du site. Si l'utilisateur est connecté, la page est dynamique et affiche les actualités récentes récupérées via l'API News.

Si l'utilisateur est connecté

2. Page de Connexion (Login)

La page de connexion permet aux utilisateurs d'accéder à leur compte en entrant leur adresse email et leur mot de passe.

3. Page d'Inscription (Signup)

La page d'inscription permet à de nouveaux utilisateurs de créer un compte en fournissant leur nom, email mot de passe et photo de profil.

4. Tableau de Bord (Dashboard)

Le tableau de bord permet de gérer les tâches avec une vue d'ensemble des tâches et des statistiques sur leur progression. Une petite icône du chatbot est disponible pour discuter et gérer les tâches via des commandes ou des conversations normales.

5. Page Profil

Sur la page profil, les utilisateurs peuvent modifier leurs informations personnelles, comme leur nom et leur mot de passe.

6. Page À Propos

La page "À propos" fournit des informations sur l'équipe de développement et les objectifs de TaskMate.

7. Page Contactez Nous

La page "Contactez Nous" permet aux utilisateurs de contacter l'équipe de support via un formulaire ou par email.

Fonctionnalités de Base

Gestion des Tâches

TaskMate vous permet de créer, organiser et suivre vos tâches de manière simple. Vous pouvez ajouter des tâches manuellement via l'interface ou utiliser le chatbot intégré pour les ajouter rapidement.

Utilisation du Chatbot

Le chatbot de TaskMate facilite la gestion des tâches par simple conversation. Il vous permet d'ajouter, marquer comme complètes et supprimer des tâches en répondant simplement aux questions posées.

Envoi de Notifications par E-mail

TaskMate envoie des notifications par e-mail en utilisant un service d'envoi d'e-mails Brevo. Ces notifications sont envoyées aux utilisateurs pour les informer des tâches proches de leur échéance ou en retard.

Fonctionnalités Avancées

API

1. Gestion des utilisateurs

1.1. Inscription

URL : http://localhost:5050/api/users/register

Méthode : POST

Body (JSON) :

{
  "name": "John Doe",
  "email": "johndoe@example.com",
  "password": "123456"
}

1.2. Connexion

URL : http://localhost:5050/api/users/login

Méthode : POST

Body (JSON) :

{
  "email": "johndoe@example.com",
  "password": "123456"
}

2. Gestion des tâches

2.1. Ajouter une tâche

URL : http://localhost:5050/api/tasks

Méthode : POST

Headers :

{
  "Authorization": "Bearer "
}

Body (JSON) :

{
  "title": "Acheter des courses",
  "description": "Acheter du lait, des œufs et du pain.",
  "dueDate": "2024-12-31",
  "priority": "High"
}

2.2. Obtenir toutes les tâches

URL : http://localhost:5050/api/tasks

Méthode : GET

2.3. Obtenir les détails d'une tâche

URL : http://localhost:5050/api/tasks/:id

Méthode : GET

2.4. Modifier une tâche

URL : http://localhost:5050/api/tasks/:id

Méthode : PUT

Body (JSON) :

{
  "title": "Acheter des fournitures",
  "description": "Acheter des stylos et des cahiers.",
  "priority": "Medium",
  "status": "Ongoing"
}

2.5. Supprimer une tâche

URL : http://localhost:5050/api/tasks/:id

Méthode : DELETE

2.6. Supprimer les tâches complétées

URL : http://localhost:5050/api/tasks/completed

Méthode : DELETE
